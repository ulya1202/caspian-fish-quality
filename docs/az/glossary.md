# Glossari (terminoloji)

Bu sənəd dissertasiyada işlədilən əsas balıq adlarının və texniki
terminlərin Azərbaycan dilində standart yazılışını və düzgün elmi
müqabillərini cəmləşdirir. **Vacib düzəliş:** *Huso huso* azərbaycanca
**bölgə** (yaxud bəzən **ağ balıq**) adlanır; **uzunburun** termini
*Acipenser nudiventris* ("ship") növünə aiddir və *Huso huso* üçün
istifadə edilməməlidir (Pourkazemi, 2006; Kottelat & Freyhof, 2007).

## Texniki terminlər

| Termin | Tərif |
|---|---|
| Sintetik məlumat | Müşahidə əvəzinə statistik modellə yaradılan rəqəmsal məlumat. |
| NORTA | NORmal-To-Anything: Qauss kopulası vasitəsilə korrelyasiyalı paylanma yaratma metodu (Cario & Nelson, 1997). |
| Kopula | Marginal paylanmaları birləşdirən birləşdirici funksiya (Sklar, 1959). |
| Kəsilmiş normal | Normaldan, lakin müəyyən [a, b] aralığına məhdudlaşdırılmış paylanma (Robert, 1995). |
| Çapraz validasiya (CV) | Modelin ümumiləşdirmə xətasını qiymətləndirmək üçün məlumatın K hissəyə bölünərək növbəli istifadə edilməsi (Stone, 1974; Kohavi, 1995). |
| Sızma (data leakage) | Test məlumatından öyrətmə zamanı yararlanma; Pipeline ilə qarşısı alınır (Kapoor & Narayanan, 2023). |
| TSTR | Train-Synthetic, Test-Real (sintetik öyrətmə, real test) protokolu (Esteban et al., 2017). |
| H-divergence proxy | Domen yaxınlığını ikili klassifikator vasitəsilə qiymətləndirmə (Ben-David et al., 2010). |
| Frobenius məsafəsi | İki matrisin elementbazaer fərqinin Frobenius normu. |
| 1-Wasserstein | İki paylanma arasında earth-mover məsafəsi (Villani, 2009). |
| AI / TI | Atherogenic / Thrombogenic Index (Ulbricht & Southgate, 1991). |
| SEM ↔ SD | SD = SEM × √n (Altman & Bland, 2005). |
