Dissertasiya §3.5.3 — sintetik məlumat + maşın öyrənməsi

Ulviyya Aliyeva

Qeyd: Bütün nəticələr sintetik məlumat üzərindədir. Canlı orqanizm üzərində təcrübə aparılmamışdır (CITES / IUCN). Demo proof-of-concept xarakterlidir.
Canlı demo aç GitHub / kod

Problem

Xəzər nərələri CITES Appendix II və IUCN siyahısında CR/EN statusundadır; birbaşa keyfiyyət təcrübələri məhduddur. Dissertasiyada metodoloji surroqat kimi Silurus glanis (naqqa balığı) və ədəbiyyat priorlarından sintetik generasiya tətbiq olunur.

Həll

  1. 6 ədəbiyyat cədvəlindən Qauss kopulası (NORTA) ilə sintetik data (N=2000)
  2. Su keyfiyyəti → zülal, lipid, kütlə proqnozu (5-fold CV, leakage-free ML)
  3. Zero-shot transfer: üç nərə növü (*A. stellatus*, *A. baerii*, *H. huso*)
Ədəbiyyat CSV → Sintetik generasiya → ML → Nərə transferi

Canlı demo

Brauzerdə slider və düymələrlə:

  • Sintetik generasiya və marginal uyğunluq
  • Su parametrləri → ət tərkibi proqnozu
  • Nərə transferi (proqnoz vs ədəbiyyat)
  • Cədvəl 3.5.6 və 3.5.7 nəticələri
