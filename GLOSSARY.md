## Technical terms

| Term | Definition |
|---|---|
| Synthetic data | Numeric data generated by a statistical model rather than direct observation. |
| NORTA | NORmal-To-Anything: copula-based correlated random vector generation (Cario & Nelson, 1997). |
| Copula | Function joining marginal distributions into a multivariate one (Sklar, 1959). |
| Truncated normal | Normal distribution restricted to [a, b] (Robert, 1995). |
| Cross-validation (CV) | K-fold partitioning to estimate generalisation error (Stone, 1974; Kohavi, 1995). |
| Data leakage | Training set contamination by test information; prevented by Pipeline (Kapoor & Narayanan, 2023). |
| TSTR | Train-Synthetic, Test-Real protocol (Esteban et al., 2017). |
| H-divergence proxy | Domain-classifier-based similarity measure (Ben-David et al., 2010). |
| Frobenius distance | Element-wise norm of matrix difference. |
| 1-Wasserstein | Earth-mover distance between two distributions (Villani, 2009). |
| AI / TI | Atherogenic / Thrombogenic Index (Ulbricht & Southgate, 1991). |
| SEM <-> SD | SD = SEM * sqrt(n) (Altman & Bland, 2005). |

## Group codes

* **AG** -- aquaculture group (controlled feeding and environment).
* **RG** -- riverine group (free-living, natural environment).
